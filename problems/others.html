Get Mean Rank Count¶

The problem is a varaint of count subarrays with average K. I’ll use the link problem for explanation.

Consider an array \(x_0, x_1, \dots, x_4\) and a subarray \(x_1, x_2, x_3\) has average \(k\). The key observation is that if every number in \(x_1, x_2, x_3\) is subtracted by \(k\), the average becomes 0, i.e.,

\[(x_1 + x_2 + x_3)/ 3 = k \implies (x_1 - k + x_2 - k + x_3 - k) / 3 = 0\]

Thus, the problem becomes that we are looking for the subarrays who have average 0. The average becomes 0 if \(x_1' + x_2' + x_3' = 0\) where \(x_i' = x_i - k\). There are two possible situations we can detect that:

  1. \(x_1' + x_2' + x_3' = 0\) directly via the sum

  2. We can rewrite \(x_1' + x_2' + x_3'\) as \((x_0' + x_1' + x_2' + x_3') - x_0'\) and \(x_0' + x_1' + x_2' + x_3' = x_0'\). In general, for \(\sum_l=i^j x_l'\), we can rewrite it as \(\sum_{l=0}^j x_l' - \sum_{l=0}^{i-1} x_l'\).

How to implement the two checks can be seen in the referenced link.
