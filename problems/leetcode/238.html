238. Product of Array Except Self¶

The intuitive solution uses prefix product and suffix product.

In order to reduce the extra space to \(O(1)\), there are two insights:

  • We can use res (the final result array) to hold prefix product. Naively, we may want to use res to hold whole productFromLeft. But, note that for the last position, we don’t need to multiply the number. For example, for nums = [1,2,3,4], we don’t need \(24\) for position 3; \(6\) is enough. In other words, productFromLeft = [1,2,6,24] but we just need [1,1,2,6] because for the final answer, we need to divide 24 by 4 anyway. In other words, res is almost same as productFromLeft but we “push” the whole array to right for one position.

  • We can use one integer to keep track of accumulated suffix product, instead of the whole array. Then, we can just multiply the accumulated suffix product into res to obtain the final answer.
